# Linguistika Frontend

Frontend React para la aplicación de gestión de horarios y tutorías.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El servidor estará disponible en `http://localhost:3000`

## Construcción

```bash
npm run build
```

## Estructura del Proyecto

- `src/components/` - Componentes reutilizables (Formularios)
- `src/pages/` - Páginas principales (Dashboard, Tutores, Cursos, etc.)
- `src/services/` - Servicios de API
- `src/App.jsx` - Componente principal con enrutamiento
- `index.html` - HTML principal

## Características

- Dashboard con estadísticas en tiempo real
- Gestión de tutores con tarifas
- Gestión de cursos por nivel
- Registr de estudiantes
- Sistema de matrículas
- Vista de tutorías programadas por día
- Cálculo y registro de pagos
